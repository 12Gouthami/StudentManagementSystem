
# Java Environment Setup – LearnTrack Project

## Java Development Kit (JDK) Version

For this project, the following Java version was used:

- **JDK Version:** Java 21 (LTS)
- **Vendor:** Oracle / OpenJDK
- **Operating System:** Windows

Java 21 is a **Long-Term Support (LTS)** version and is stable, secure, and suitable for learning Core Java concepts.

---

## Steps to Install Java (JDK)

1. Go to the official Oracle Java website:  
   https://www.oracle.com/java/technologies/downloads/

2. Download the **JDK 21** installer for your operating system (Windows x64).

3. Run the installer and follow the on-screen installation instructions.

4. After installation, verify Java by running the following command in Command Prompt or PowerShell:

```bash
java -version
```

## Hello World Program Execution

To verify that Java was installed and working correctly, a simple **Hello World** program was created, compiled, and executed.

### Program Code
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```


The program was written in a file named `HelloWorld.java`. It was then compiled using the Java compiler, which converts the source code into bytecode. After successful compilation, the program was executed using the Java command.

When the program was run, the message **“Hello, World!”** was displayed on the console.  
This confirms that Java is properly installed and that programs can be compiled and executed successfully.
